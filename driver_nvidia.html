Instalación del controlador NVIDIA



CódigoCristo / Marzo 17, 2023

Con el siguiente comando nos dará información de nuestra tarjeta gráfica

[root@nombre_de_pc ~]# lspci | grep VGA

En el primer ejemplo estoy usando una Nvidia GT 610 usando lspci



En este segundo ejemplo estoy usando una Nvidia RTX 2080 usando neofetch



Puede buscar en la base de datos de paquetes una lista completa de controladores de video de código abierto:

[root@nombre_de_pc ~]# pacman -Ss xf86-video

El kernel de Linux incluye controladores de video de código abierto y soporte para framebuffers acelerados por hardware. Sin embargo, se requiere soporte optimizado para OpenGL y aceleración 2D/3D en Xorg o Wayland


Por ese motivo es necesario instalar un controlador adecuado de código abierto o código cerrado

● Driver = incluye aceleración 2D o 3D ● OpenCL = Uso de aplicaciones con la GPU ● OpenGL o Vulkan = producen gráficos 2D y 3D ● VDPAU GPU = Decodifica y codifica video con la GPU



Si no puede encontrar el controlador específico instalado para el hardware, pruebe instalando un controlador genérico, que maneja una gran cantidad de chips pero no incluye ninguna aceleración 2D o 3D asegurando que tendra video si o si sea con cualquier hardware que tenga video

[root@nombre_de_pc ~]# pacman -S xf86-video-vesa xf86-video-fbdev

Información

Arch Linux, una distribución de Linux® liviana y flexible que intenta mantenerlo simple. Actualmente tenemos paquetes oficiales optimizados para la arquitectura x86-64. Complementamos nuestros conjuntos de paquetes oficiales con un repositorio de paquetes operado por la comunidad que crece en tamaño y calidad todos los días.

Noticias Recientes
